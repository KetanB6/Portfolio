○ Student Management System

I developed a console-based Student Management System in Core Java to efficiently
manage student details in schools and colleges. This system handles student records,
fee management, attendance, exam reports, and library operations.

Key Features

Student Management
1. Fee Structure – Displays the fee details for students.
2. Add Student – Registers a new student.
3. Show All Students – Lists all students in the selected department.
4. Search Student – Finds a student using their roll number.
5. Attendance Records – Maintains student attendance.
6. Update Student Info – Allows modifications to student details.
7. Due Fee Payment – Tracks and processes pending fees.
8. Exam Report – Stores and displays student marks.

Library Management
1. Add Book – Adds new books to the library.
2. Search Book – Finds books using keywords or IDs.
3. Lend Book – Issues books to students.

The library and student management systems are interconnected—students can use
their roll numbers to borrow books, ensuring seamless integration between both
modules.

This project helped me understand how enterprise software works and how different
modules within a system can be efficiently developed and connected.
