Padarth is a web-based ingredient analysis tool that allows users to scan,
upload, and search for ingredients to retrieve detailed information and know
how safe an ingredient is for health.
Features
1. Scan Ingredient – Capture an image using a camera, extract text, and get ingredient details.
2. Upload Ingredient – Upload an image directly for analysis.
3. Search Ingredient – Search for ingredients by name.
How It Works
1. The image is captured/uploaded and converted to Base64.
2. The image is processed to extract text.
3. Extracted text is sent to Gemini AI for ingredient details.
4. Processed data is sent to JSP for display.
Technologies Used
Frontend – HTML, CSS, JavaScript
Backend – Java Servlet
UI Rendering – JSP (Java Server Pages)
Throughout this project development, I learned a lot, including Servlets,
JSP, and API integration.
